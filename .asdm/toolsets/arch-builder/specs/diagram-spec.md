# 架构图规范

## 1. 图表类型

| 类型 | 用途 | ViewBox |
|------|------|---------|
| 系统架构图 | 整体系统分层 | `0 0 960 700` |
| 组件图 | 模块依赖关系 | `0 0 960 600` |
| 数据流图 | 数据流转路径 | `0 0 960 600` |
| 部署图 | 部署拓扑 | `0 0 1200 600` |
| 序列图 | 交互时序 | 动态计算 |

## 2. 分层规范 (系统架构图)

```
┌──────────────────────────────────────┐
│  IM Channels (Slack/Telegram/...)    │  ← 接入层
├──────────────────────────────────────┤
│  HTTP API (chi router)               │  ← API 层
├──────────────────────────────────────┤
│  Business Layer (DDD Bounded Context)│  ← 业务层
├──────────────────────────────────────┤
│  Infrastructure Layer                │  ← 基础设施层
├──────────────────────────────────────┤
│  Data Storage                        │  ← 数据层
└──────────────────────────────────────┘
```

## 3. 颜色规范

| 组件类型 | 颜色 | 说明 |
|----------|------|------|
| 接入层 | `#3b82f6` (蓝) | 外部入口 |
| API 层 | `#8b5cf6` (紫) | 路由网关 |
| 业务层 | `#10b981` (绿) | 核心业务 |
| 基础设施 | `#f59e0b` (橙) | 共享组件 |
| 数据层 | `#ef4444` (红) | 存储系统 |
| 外部服务 | `#6b7280` (灰) | 第三方 |

## 4. 箭头规范

| 流向 | 颜色 | 宽度 | 样式 |
|------|------|------|------|
| 请求流 | `#2563eb` | 2px | 实线 |
| 数据流 | `#059669` | 1.5px | 实线 |
| 控制流 | `#ea580c` | 1.5px | 实线 |
| 异步流 | `#6b7280` | 1.5px | 虚线 |

## 5. 字体规范

- 标题: 16-18px, bold
- 组件标签: 13-14px, normal
- 子标签: 11-12px, normal
- 图例: 11px, normal

## 6. 输出规范

- SVG: 主要格式，支持 Web 渲染
- PNG: 2x 分辨率，用于文档嵌入
- 命名: `{项目名}-{图表类型}-v{版本}.{svg|png}`
